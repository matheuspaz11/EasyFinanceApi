## Sobre o projeto
Esta API foi desenvolvida para simplificar o gerenciamento de despesas pessoais. Permite uma visualização das informações mais importantes sobre despesas mensais, como vencimento, valor e status. 

## Estrutura de pastas

- **EasyFinanceApi**  
  - **Context**  
  - **Controllers**  
  - **Helpers**  
    - Util  
    - Validators  
  - **Mapper**  
  - **Migrations**  
  - **Models**  
    - DTOs  
    - Entities  
    - Enums  
  - **Repository**  
    - Interfaces  
  - **Services**  
    - Interfaces
